elasticsearch是面向文档(document oriented)的 这样意味着它可以存储整个对象或文档(document) 然而它不仅仅是存储 还会索引(index)每个文档的内容使之可以被搜索 在Elasticsearch中 你可以对文档(而非成行成列的数据)进行索引 搜索 排序 过滤 这种理解数据的方式与以往完全不同 这也是Elasticsearch能够执行复杂的全文搜索的原因之一 JSON Elasticsearch使用javascript对象符号(JavaScript Object Notation) 作为文档序列化格式 JSON现在已经被大多语言所支持 而且已经成为NoSQL领域的标准格式

Elasticsearch介绍 (参才文档https://es.xiaoleilu.com/) ------------------------------------------------------------------ Elasticsearch 是一个基于Apache的开源搜索引擎 lucenc可以被认为是迄今为止最先进,性能最好的,功能最全的搜索引擎库 Elasticsearch 分布式的实时文件存储 每个字段都被索引并可被搜索 分布式的实时文件搜索引擎 可以扩展到上百台服务器 处理PB级结构化或非结构化数据 安装elasticsearch(参考elc-install.html) 安装marvel ------------------------------------------------------------------ marvel是Elasticsearch的管理和监控工具 elasticsearch5使用x-pack bin/plugin -i elasticsearch/marvel/latest 安装x-pack elasticsearch ---> bin/elasticsearch-plugin install x-pack kibana ---> bin/kibana-plugin install x-pack 运行Elasticsearch ------------------------------------------------------------------ su apeng bin/elasticsearch curl http://192.168.233.132:9200/_search?pretty=true # 使用x-pack路由没有直接权限需要使用 -u username missing authentication token for REST request [/_search?pretty=true] curl -u elastic http://192.168.233.132:9200/_search?pretty=true # 会显示所有索引 也可以使用浏览器访问 http://192.168.233.132:9200/_search?pretty=true 分布式的特性 ------------------------------------------------------------------ Elasticsearch 为分布式而生 它的设计隐藏了分布式本身的复杂性 Elasticsearch 在分布式概念上做了很大程序上的透明化 可以不去知道分布式系统 分片 集群发现或者其他大量的分布式概念 将你的文档分区到不同的容器或者分片(shards)中 它们可以存在于一个或多个节点中 将分片均匀的分配到各个节点 对索引和搜索做负载均衡 冗余每一个分片 防止硬件故障造成的数据丢失 将集群中任意一个节点上的请求路由到相应数据所在的节点 无论是增加节点 还是移除节点 分片都可以做到无缝的扩展和迁移
